รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-กรุงไทยร่วมกับกระทรวงการคลังให้บริการ QR Code ที่ศูนย์อาหารทำเนียบรัฐบาล







                                วันอังคารที่ 26 กันยายน 2560                              





    179  





      พิมพ์    
     
    
















   กรุงไทยร่วมกับกระทรวงการคลังให้บริการ QR Code ที่ศูนย์อาหารทำเนียบรัฐบาล
 

   ในวันนี้ นายกรัฐมนตรี เปิดงาน “Thailand 4.0 Digital Showcase by MOF”  เพื่อนำเทคโนโลยีการชำระเงินผ่าน QR Code และแอปพลิเคชั่นสำหรับร้านค้า  มานำเสนอ ครม. สำหรับใช้ชำระค่าอาหารและเครื่องดื่มแทนเงินสดที่ศูนย์อาหารของทำเนียบรัฐบาล 





   ในวันนี้ (26 กันยายน 2560) พลเอกประยุทธ์ จันทร์โอชา นายกรัฐมนตรี เป็นประธานเปิดงาน“Thailand 4.0 Digital Showcase by MOF” โดยมี นายอภิศักดิ์ ตันติวรวงศ์ รัฐมนตรีว่าการกระทรวงการคลัง ดร.สมชัย สัจจพงษ์ ปลัดกระทรวงการคลัง และ นายผยง ศรีวณิช กรรมการผู้จัดการใหญ่ ธนาคารกรุงไทย ร่วมให้การต้อนรับ เพื่อนำเทคโนโลยีการชำระเงินผ่าน QR Code และแอปพลิเคชั่นสำหรับร้านค้า มานำเสนอคณะรัฐมนตรี สำหรับใช้ชำระค่าอาหารและเครื่องดื่มแทนเงินสดที่ศูนย์อาหารของทำเนียบรัฐบาล

ดร.สมชัย สัจจพงษ์ ปลัดกระทรวงการคลัง และประธานกรรมการธนาคารกรุงไทย เปิดเผยว่า กระทรวงการคลังดำเนินตามนโยบาย Thailand 4.0 ที่มุ่งหวังในการวางโครงสร้างเพื่อเศรษฐกิจและสังคมระยะยาวของประไทยที่ขับเคลื่อนด้วยเทคโนโลยี นวัตกรรม เพื่อความยั่งยืน ภายใต้สโลแกน “มั่นคง มั่งคั่ง และยั่งยืน” โดยหนึ่งในวิสัยทัศน์เชิงนโบบายของรัฐบาลต้องการให้ประเทศไทยเป็นสังคมไร้เงินสด หรือ Cashless Society

ทั้งนี้ กระทรวงการคลังได้นำเอาเทคโนโลยี QR code มาใช้ให้เกิดเป็นรูปธรรม เพื่อความสะดวก และปลอดภัย ในการชำระเงินทั้งลูกค้า และผู้ประกอบการ ตามวิสัยทัศน์ของกระทรวงการคลังตามนโยบายคลัง 4.0 คือ การเป็นเสาหลักทางเศษรฐกิจ เพื่อการพัฒนา ที่ยั่งยืน ซึ่งหนึ่งในยุทธศาสตร์หลักตามวิสัยดังกล่าว คือ การพัฒนาศักยภาพ และเพิ่มขีดความสามารถในการแข่งขัน

นายผยง ศรีวณิช กรรมการผู้จัดการใหญ่ ธนาคารกรุงไทย เปิดเผยว่าธนาคารได้นำเอาเทคโนโลยี KRUNGTHAI QR Code มาให้ข้าราชการ พนักงานและลูกจ้างของทำเนียบรัฐบาลได้ใช้งาน ณ ศูนย์อาหารของทำเนียบรัฐบาล  โดยการทำธุรกรรมชำระเงินค่าอาหารและเครื่องดื่มผ่าน KRUNGTHAI QR Code บนแอปพลิเคชั่น KTB netbank ทำให้การใช้จ่ายในชีวิตประจำวันสะดวก และง่ายขึ้น ด้านผู้ขายเพียงโหลดแอปพลิเคชั่น “เป๋าตุง” สมัครพร้อมเพย์และติด QR Code ไว้ที่ร้าน ก็จะได้รับเงินเข้าบัญชีทันที ซึ่งลดความยุ่งยากในการรับ/ทอนเงิน ด้านผู้ซื้อก็ไม่ต้องพกเงินสด เพียงแค่สแกน QR Code ของร้านค้า ระบุจำนวนเงิน และ PIN เพื่อยืนยันการทำรายการ เพียงเท่านี้ก็ชำระเงินเรียบร้อย

ที่ผ่านมาได้มีการเปิดตัว MOF Digital Canteen ณ โรงอาหารของกระทรวงการคลัง พร้อมติดตั้ง KRUNGTHAI QR Code และแนะนำวิธีการใช้งาน แอปพลิเคชั่น “เป๋าตุง” ให้กับร้านค้าต่างๆ ที่ตลาด อตก. ตลาดสามชุก จ.สุพรรณบุรี  และตลาดกรุงศรี จ.พระนครศรีอยุธยา และมีแผนจะขยายการให้บริการ KRUNGTHAI QR Code ไปยังร้านค้าอื่นๆ อย่างต่อเนื่อง เพื่อให้ร้านค้าและประชาชนทั่วไปมีโอกาสเข้าถึงเทคโนโลยีใหม่ๆ ได้อย่างง่ายดาย เพื่อตอบสนองนโยบายของรัฐในการลดการใช้เงินสด
    
ธนาคารกรุงไทยพร้อมเป็นอีกหนึ่งกลไกในการสนับสนุนนโยบายคลัง 4.0 เพื่อร่วมขับเคลื่อนนโยบายของประเทศ โดยการพัฒนาผลิตภัณฑ์และบริการทางการเงินใหม่ๆ เพื่อให้ประชาชนทุกคนมีโอกาสเข้าถึงบริการทางการเงินที่สะดวกและปลอดภัยอย่างทั่วถึง เพื่อลดความเหลื่อมล้ำทางเศษรฐกิจและเป็นรากฐานที่สำคัญสู่การพัฒนาประเทศอย่างยั่งยืน
